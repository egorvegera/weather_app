# weather_app

A new Flutter project.

# Функционал
Приложение берет данные о погоде из OpenWeather;
- На экране Профиль реализована возможность зарегистрироваться и залогиниться по Email/password или анонимно через Firebase;
  Иконка справа вверху. При нажатии появится модальное окно;
- На экране Профиль реализована возможность опубликовать пост с комментарием о погоде;
  Для этого нужно ввести название города, получить данные о погоде, написать комментарий и нажать "опубликовать пост";
- На экране "Настройки" (иконка появляется вверху справа после аутентификации) реализована возможность сменить аватарку, загурзив ее с устройства;
- Опубликованные посты отображаются в "Ленте";
- Ссылки на экраны в нижней навигационной панели.
